核心架构解析

一期采用最简高可用架构，旨在将复杂深度学习推理与高并发 API 层隔离。拒绝过度设计，专注于提供稳定的本地化异步任务能力。

系统已完全脱离旧有 OpenFang/Shipinbot 运行时路径进行独立部署和解耦。

  • FastAPI 前台：负责接受上传、校验 API Key、写入 SQLite 投递任务以及幂等处理。
  • Worker 后台轮询：抢占式并发任务锁处理，具备状态更新与回调自动重试机制。
  • Provider 适配器：支持 `comfy_diffueraser` (AI 主链) 和 `local_fallback` (FFmpeg 复制/保活)，内置完善降级链。
